Streamlit To-Do App 

A simple, interactive To-Do List application built with Python and Streamlit.
Add, edit, delete, and mark tasks as done. Tasks persist during the session using Streamlit's session state.

Features

Add new tasks with a single click.

Mark tasks as done or undone with checkboxes.

Edit task text easily.

Delete tasks individually.

Clear all completed tasks.
